Aide : xmcontact

Déscription


« xmcontact » est un module de contact. Vous pouvez créer des catégories qui permettent de prendre contact directement avec un département, un service, une personne, etc. La page d'index peut être configurée directement depuis les préférences.

Installer / Désinstaller


Aucune mesure spéciale n'est nécessaire, suivez la procédure d'installation standard – extraire le dossier /contact vers le dossier ../modules. Installez le module par Admin -> System Module -> Modules.

Des instructions détaillées sur comment installer un module sont disponibles sur le Manuel des opérations de XOOPS

Mode d'emploi


Il n'y a vraiment rien à faire sur l'administration du site, mise à part configurer les préférences.
